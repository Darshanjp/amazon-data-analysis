# 🛒 Amazon Product Data Analysis

## 📌 Project Overview
Exploratory Data Analysis (EDA) and clustering on Amazon product data to uncover insights on pricing, discounts, and category performance.

## 🛠️ Tools & Technologies
- **Python** — Pandas, NumPy
- **Visualization** — Matplotlib, Seaborn
- **Machine Learning** — Scikit-learn (KMeans Clustering, StandardScaler)

## 📊 Key Analysis Performed
- Top 10 most expensive products
- Category with highest average price
- Number of products per category (bar chart)
- Average discount percentage per category
- Correlation heatmap between price, quantity, and discount
- Distribution of product prices (KDE plot)
- KMeans clustering to segment products by price and quantity

## 📁 Dataset
- `Amazon.csv` — Contains product details including category, price, quantity, discount, and total amount

## 📈 Visualizations
- Bar Chart — Products per Category
- Heatmap — Correlation between Price, Quantity, Discount
- KDE Plot — Price Distribution
- Cluster Plot — Product Segmentation

## 🔍 Insights
- Identified top-performing and overpriced product categories
- Found categories with highest average discounts
- Segmented products into clusters for pricing strategy analysis

pip install pandas numpy matplotlib seaborn scikit-learn
